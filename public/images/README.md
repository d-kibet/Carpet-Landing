## Image Optimization Tips
- Use WebP format for better compression and quality
- Optimize images before upload using tools like TinyPNG or Squoosh
- Target file size: 200-500KB per image for web performance
- Ensure proper aspect ratios to avoid distortion
- Use high-quality images as they appear with opacity overlays
